== Description ==
A plugin that will help you remove special characters from the name of your files.

= Contribute =
You can contribute to the source code in our [GitHub](https://github.com/douglasfaria10/Remove-Special-Characters-Upload) page.

== Installation ==
Just install and enjoy plugin. This plugin works with the native upload function.

== Frequently Asked Questions == 
= This plugin works with others plugins that use upload function?
If this plugin to use native upload function, yes.
